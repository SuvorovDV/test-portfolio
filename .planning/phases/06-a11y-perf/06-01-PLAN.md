<task type="auto">
  <name>Task 2: Focus-visible enhancement</name>
  <files>src/styles/globals.css</files>
  <read_first>src/styles/globals.css</read_first>
  <action>
  В `src/styles/globals.css` текущий focus-visible:
  ```css
  :focus-visible {
    outline: 2px solid var(--phosphor);
    outline-offset: 2px;
  }
  ```

  Улучшения:
  - Добавить `outline-style: solid` explicit (некоторые браузеры render'ят 'auto' стиль)
  - Для элементов inside inverted контекста (активная навбар-ссылка на phosphor фоне) — focus-ring будет невидим. Нужно условное правило:
    ```css
    nav[aria-label='Primary'] a[aria-current='location']:focus-visible {
      outline-color: var(--bg);
    }
    ```
    НО текущий TerminalPrompt рендерит это — правило должно быть в TerminalPrompt.module.css.

  Замена в TerminalPrompt.module.css — добавить к существующему `.command:hover, .command:focus-visible`:
  ```css
  .command[aria-current='location']:focus-visible {
    outline-color: var(--bg);
  }
  ```

  Аналогично — ссылки в hover/focus-visible в globals.css уже инвертированы. Проверить что outline корректно видно ПРИ hover: текущий правило:
  ```css
  a:hover, a:focus-visible {
    background-color: var(--phosphor);
    color: var(--bg);
    outline: none;
  }
  ```
  `outline: none` — это **анти-паттерн**. Надо: при hover OK убрать (мы инвертировали фон и это сам по себе focus-indicator), но при :focus-visible — **оставить** outline, иначе keyboard пользователь теряет focus trail.

  Исправление в globals.css:
  ```css
  a:hover {
    background-color: var(--phosphor);
    color: var(--bg);
    text-decoration: none;
  }

  a:focus-visible {
    outline: 2px solid var(--phosphor);
    outline-offset: 2px;
  }
  ```

  Отделили hover и focus-visible — корректнее.
  </action>
  <verify>Tab по навбару: активная команда (инверсия) показывает outline в цвете bg (не phosphor который сольётся). Обычная ссылка показывает phosphor outline.</verify>
  <acceptance_criteria>
    - globals.css: `a:hover` и `a:focus-visible` разделены, hover не убирает outline
    - TerminalPrompt.module.css: добавлено правило для focus-visible на inverted active link
    - build/lint/typecheck зелёные
  </acceptance_criteria>
