  <action>
  Создать `src/components/Cursor.module.css`:

  ```css
  .cursor {
    display: inline-block;
    width: 0.6ch;
    height: 1em;
    background-color: currentColor;
    vertical-align: text-bottom;
    margin-left: 0.15ch;
    /* 1 Hz — безопасно для WCAG (< 3 Hz) */
    animation: blink 1s steps(2) infinite;
  }

  .cursorStatic {
    display: inline-block;
    width: 0.6ch;
    height: 1em;
    background-color: currentColor;
    vertical-align: text-bottom;
    margin-left: 0.15ch;
    /* Без анимации — статичный блок */
  }

  @keyframes blink {
    0%,
    50% {
      opacity: 1;
    }
    50.01%,
    100% {
      opacity: 0;
    }
  }

  @media (prefers-reduced-motion: reduce) {
    .cursor {
      animation: none;
    }
  }
  ```

  ПРИМЕЧАНИЕ: используем `currentColor` — курсор наследует цвет у родителя (в h2 — ярко-зелёный, в p — dim-green). Это даёт визуальную консистентность и автоматически переключается при смене палитры.

  Два класса (.cursor и .cursorStatic) — избыточны, достаточно одного `.cursor` с media query. Оставим только `.cursor`:

  **Финальная версия:**
  ```css

    display: inline-block;
    width: 0.6ch;
    height: 1em;
    background-color: currentColor;
    vertical-align: text-bottom;
    margin-left: 0.15ch;
    animation: blink 1s steps(2) infinite;
  }

  @keyframes blink {
    0%,
    50% {
      opacity: 1;
    }
    50.01%,
    100% {
      opacity: 0;
    }
  }

  @media (prefers-reduced-motion: reduce) {
    .cursor {
      animation: none;
    }
  }
  ```
