  <files>src/components/CRTShell.module.css</files>
  <read_first>.planning/phases/02-crt-theming/CONTEXT.md §Scanlines, §CRT Vignette, §Flicker/Glow — особенно D-17 (частота!)</read_first>
  <action>
  Создать `src/components/CRTShell.module.css`:

  ```css
  .shell {
    position: relative;
    min-height: 100vh;
  }

  /* Scanlines — повторяющийся градиент, фиксирован к viewport */
  .scanlines {
    position: fixed;
    inset: 0;
    pointer-events: none;
    z-index: 100;
    background-image: repeating-linear-gradient(
      to bottom,
      transparent 0,
      transparent calc(var(--scanline-period) - 1px),
      var(--scanline-color) calc(var(--scanline-period) - 1px),
      var(--scanline-color) var(--scanline-period)
    );
    opacity: var(--scanline-opacity);
    /* Мягкий drift — 2 Hz безопасно для WCAG 2.3.1 (< 3/s) */
    animation: scanline-drift 0.5s linear infinite;
    will-change: background-position;
  }

  @keyframes scanline-drift {
    from {
      background-position-y: 0;
    }
    to {
      background-position-y: var(--scanline-period);
    }
  }

  /* Vignette — радиальный градиент + inset box-shadow для углов */
  .vignette {
    position: fixed;
    inset: 0;
    pointer-events: none;
    z-index: 101;
    background: radial-gradient(
      ellipse at center,
      transparent 50%,
      var(--vignette-intensity) 100%
    );
    box-shadow: inset 0 0 150px 30px var(--vignette-intensity);
  }

  /* Reduced motion — останавливаем drift, оставляем сам scanline паттерн (статичный) */
  @media (prefers-reduced-motion: reduce) {
    .scanlines {
      animation: none;
    }
  }
  ```

  КРИТИЧНО:
  - Частота 2 Hz (0.5s period) — ниже WCAG 2.3.1 порога 3 Hz
  - НЕ использовать opacity-flicker (D-17 решение в CONTEXT.md)
  - z-index: scanlines=100, vignette=101 — vignette рисуется поверх scanlines (иначе scanlines заметны на краях где контента нет)
  - `will-change: background-position` — GPU-acceleration hint

  НЕ добавлять:
  - curvature через `transform: perspective()` (ломает layout)
  - SVG-фильтры (cost > value)
  </action>
  <verify>Файл компилируется как CSS module; ключевой кадр scanline-drift объявлен</verify>
  <acceptance_criteria>
    - .scanlines класс имеет `pointer-events: none`, `position: fixed`, `inset: 0`

    - `@media (prefers-reduced-motion: reduce)` отключает анимацию
    - .vignette рендерится поверх .scanlines (z-index выше)
    - НЕТ `transform: perspective`, НЕТ SVG-фильтров
