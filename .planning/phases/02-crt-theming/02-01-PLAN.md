  Создать `src/styles/tokens.css`:

  ```css
  :root {
    /* Palette — phosphor green (меняется на amber заменой двух переменных) */
    --phosphor: #33ff33;
    --phosphor-dim: #1fa81f; /* ≈70% luminance */
    --bg: #0a0f0a;

    /* Scanlines */
    --scanline-color: rgba(0, 0, 0, 0.35);
    --scanline-opacity: 0.1;
    --scanline-period: 3px;

    /* Glow */
    --glow-blur-heading: 4px;
    --glow-blur-body: 2px;

    /* Vignette */
    --vignette-intensity: rgba(0, 0, 0, 0.8);

    /* Typography */
    --font-mono: 'VT323', 'Courier New', monospace;
    --fs-base: 18px;
    --fs-base-desktop: 20px;
    --lh-base: 1.35;

    /* Spacing (ch-based — tied to font) */
    --space-1: 0.5ch;
    --space-2: 1ch;
    --space-3: 2ch;
    --space-4: 4ch;
    --space-5: 8ch;
  }
  ```

  Никаких других правил в этом файле — только `:root { ... }`. Это источник истины для палитры.

<task type="auto">
  <name>Task 3: src/styles/globals.css — reset + base</name>
  <files>src/styles/globals.css</files>
  <read_first>src/styles/tokens.css, .planning/research/PITFALLS.md §Performance (шрифты) и §Accessibility (контраст)</read_first>
  <action>
  Создать `src/styles/globals.css`:

  ```css
  @import './tokens.css';

  /* Reset — минимальный, не modern-normalize */
  *,
  *::before,
  *::after {
    box-sizing: border-box;
    margin: 0;
    padding: 0;
  }

  html {
    font-size: var(--fs-base);
    line-height: var(--lh-base);
    /* Плавный scroll у анкоров — хук useHashNavigation его переопределяет для reduced-motion */
    scroll-behavior: smooth;
  }

  @media (min-width: 768px) {
    html {
      font-size: var(--fs-base-desktop);
    }
  }

  @media (prefers-reduced-motion: reduce) {
    html {
      scroll-behavior: auto;
    }
    *,
    *::before,
    *::after {
      animation-duration: 0.001ms !important;
      animation-iteration-count: 1 !important;
      transition-duration: 0.001ms !important;
    }
  }

  body {
    background-color: var(--bg);
    color: var(--phosphor-dim);
    font-family: var(--font-mono);
    min-height: 100vh;
    /* Фосфорный glow для body — слабее чем у заголовков */
    text-shadow: 0 0 var(--glow-blur-body) var(--phosphor-dim);
    /* Мягкое сглаживание */
    -webkit-font-smoothing: antialiased;
  }

  /* Заголовки — ярче, жирнее glow */
  h1,
  h2,
  h3 {
    color: var(--phosphor);
    text-shadow: 0 0 var(--glow-blur-heading) var(--phosphor);
    font-weight: normal; /* VT323 не имеет bold-варианта */
    letter-spacing: 0.02em;
  }

  h1 {
    font-size: 2rem;
    margin-bottom: var(--space-3);
  }
  h2 {
    font-size: 1.6rem;
    margin-bottom: var(--space-3);
  }

  p {
    margin-bottom: var(--space-3);
    max-width: 75ch;
  }

  /* Ссылки — подчёркнутые, как в терминале/man */
  a {
    color: var(--phosphor);
    text-decoration: underline;
    text-underline-offset: 0.15em;
    text-decoration-thickness: 1px;
  }

  a:hover,
  a:focus-visible {
    background-color: var(--phosphor);
    color: var(--bg);
    text-decoration: none;
    outline: none;
  }

  /* Focus ring — инверсия, видна всем */
  :focus-visible {
    outline: 2px solid var(--phosphor);
    outline-offset: 2px;
  }

  /* Списки — без маркеров (навбар рендерит команды, не буллеты) */
  ul,
  ol {
    list-style: none;
  }

  /* Секции — вертикальный ритм */
  section {
    padding: var(--space-5) var(--space-4);
    min-height: 100vh; /* Якорь-навигация ведёт к полному экрану */
  }

  /* Навбар — sticky top */
  nav[aria-label='Primary'] {
    position: sticky;
    top: 0;
    z-index: 10;
    background-color: var(--bg);
    padding: var(--space-2) var(--space-4);
    border-bottom: 1px solid var(--phosphor-dim);
  }

  nav[aria-label='Primary'] ul {
    display: flex;
    gap: var(--space-4);
    flex-wrap: wrap;
  }

  nav[aria-label='Primary'] a[aria-current='location'] {
    background-color: var(--phosphor);
    color: var(--bg);
    text-decoration: none;
  }

  /* Main — ограничиваем ширину для читаемости */
  main {
    max-width: 100ch;
    margin: 0 auto;
  }
  ```

  Ограничения:
  - НЕ включать scanlines/vignette здесь (это 02-02)
  - НЕ включать Cursor-стили здесь (это 02-03)
  - Reset минимальный — избегаем box-sizing дублей с tokens.css
  - `font-weight: normal` на заголовках важно — VT323 не имеет bold
  </action>
