  <files>public/favicon.svg</files>
  <action>
  Создать `public/favicon.svg` — простой CRT-блок-курсор на чёрном фоне:

  ```xml
  <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 32 32">
    <rect width="32" height="32" fill="#0a0f0a"/>
    <rect x="10" y="8" width="12" height="16" fill="#33ff33">
      <animate attributeName="opacity" values="1;0;1" dur="1s" repeatCount="indefinite"/>
    </rect>
  </svg>
  ```

  Мигающий phosphor-курсор. Частота мигания 1 Hz — соответствует нашему Cursor в приложении.

  ВАЖНО: SMIL animate (`<animate>`) поддерживается в большинстве браузеров для favicon. Если хочется совсем без анимации — убрать `<animate>` элемент.
  </action>
  <verify>Файл создан; после build в dist/ появится</verify>

  </acceptance_criteria>

  ```html

      <!-- Open Graph -->
      <meta property="og:type" content="website" />
      <meta property="og:url" content="https://suvorovdv.github.io/test-portfolio/" />
      <meta property="og:title" content="Suvorov Denis — full-stack developer" />
      <meta property="og:description" content="Personal portfolio — web apps and Telegram bots (Python, TypeScript, aiogram)." />
      <meta property="og:image" content="https://suvorovdv.github.io/test-portfolio/og.svg" />
      <meta property="og:image:width" content="1200" />
      <meta property="og:image:height" content="630" />

      <!-- Twitter Card -->
      <meta name="twitter:card" content="summary_large_image" />
      <meta name="twitter:title" content="Suvorov Denis — full-stack developer" />
      <meta name="twitter:description" content="Personal portfolio — web apps and Telegram bots." />
      <meta name="twitter:image" content="https://suvorovdv.github.io/test-portfolio/og.svg" />

      <meta name="theme-color" content="#0a0f0a" />
    </head>
  ```

  ВАЖНО:
  - `<link rel="icon">` путь использует `/test-portfolio/favicon.svg` — абсолютный с base. Vite для `href` в index.html НЕ переписывает — поэтому жёстко прописываем полный путь.
  - `og:image` — абсолютный URL, ОБЯЗАТЕЛЬНО с https и доменом (соцсети не принимают относительные)
  - `theme-color` — цвет мобильной адресной строки, phosphor-theme соответствующий
  - Порядок тегов не критичен, но `<title>` и `<meta charset>` должны быть в первых 1024 байтах для корректного парсинга SSR-ботов (мы — SPA, но всё равно best practice)

  НЕ добавлять:
